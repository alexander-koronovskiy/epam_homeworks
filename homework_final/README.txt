Вариант N: работа с внешним API

Необходимо разработать приложение для взаимодействия с внешним API: https://randomuser.me/.
Основная страница приложение представляет из себя простую web страницу
с полем для ввода количества человек, которые необходимо загрузить с API,
а также таблица с информацией о случайно выбранном человеке.

Требования:
Хранить только информацию о: пол человека, имя, фамилию, номер телефона, email, место проживания.
При перезагрузке страницы таблица с информацией обновляется случайным образом.
Вся информация о людях хранится в базе данных. При запросе от клиента каждый раз выдаётся случайный человек из базы данных.
Использовать любой веб фреймворк и базу данных (выбор необходимо будет обосновать)
Когда сервер стартует, то он должен загрузить 100 случайных людей из API.
выполнение программы должно происходить внутри docker контейнера
Интерфейс должен выглядеть более или менее приятно
Покрыть функциональность тестами в разумных пределах.
(*) Интерфейс должен иметь кнопку, по нажатию на которую происходит обновление информации в таблице без перезагрузки исходной страницы.
(*) Можно получить информацию о конкретном человеке обратившись к нему по адресу: http://homepage/user_id
(*) Интерфейс должен иметь дополнительную кнопку, по которой можно загрузить информацию о всех людях, содержащихся в БД в виде JSON файла.
