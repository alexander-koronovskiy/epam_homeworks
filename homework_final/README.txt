Работа с внешним API (Вариант N)

Разработано приложение для взаимодействия с внешним API: https://randomuser.me/
Основная страница приложения представляет из себя простую web страницу
с полем для ввода количества человек, которые необходимо загрузить с API,
а также таблица с информацией о случайно выбранном человеке.

Функционал:
Храниться следующая информация: пол человека, имя, фамилия, номер телефона, email, место проживания.
При перезагрузке страницы таблица с информацией обновляется случайным образом.
Вся информация о людях хранится в базе данных.
При запросе от клиента каждый раз выдаётся случайный человек из базы данных.
Используется Flask, Sql lite 3, pewee ORM
Когда сервер стартует, то он должен загрузить 100 случайных людей из API.
выполнение программы происходит внутри docker контейнера
Интерфейс выглядит более или менее приятно

Функциональность пока не покрыта тестами.

(*) Интерфейс должен иметь кнопку, по нажатию на которую происходит обновление информации
в таблице без перезагрузки исходной страницы.
(*) Можно получить информацию о конкретном человеке обратившись к нему по адресу: http://homepage/user_id
(*) Интерфейс должен иметь дополнительную кнопку,
по которой можно загрузить информацию о всех людях, содержащихся в БД в виде JSON файла.
